# ? ) FAQ
### Q. アドオンを入れても何も起きません。
> ワールド設定で実験項目の **Beta-API** がオンになっていることを確認してください。
<br>

### Q. ERRORが出て、何も起きませんでした。
> 動作が正常に実行されなかった可能性があります。
>
> 詳しくは[ErrorList](https://github.com/Apedy/WorldEdit/blob/master/ErrorList.md)を確認してください。
<br>
